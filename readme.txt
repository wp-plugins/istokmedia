istOK MEdia Bannerwerbung für Deutsches ClickBank zum Geld Verdienen (wie früher Digibux)

== Description ==

Warum Cent Beträge auf die Klicks verdienen wenn man bei diesem Modell, direkt anteilmässig von den hohen Clickbank Ausschüttungen profitieren kann, Testen Sie es 2-3 Monate und sehen selbst. Sie müssen sich nicht um die Wasserzeichen versehenen und auffindbaren Banner kümmern, auch nicht um die Programm Aktualisierungen, Starts und Endungen. Über 500 Werbe Banner werden jetzt im System rotiert.

Sie werden automatisch über Einblendungen, Klicks und Verkäufe informiert.

Bitte als Partner (Publisher) bei istOK MEdia registrieren.

Danach Einlogen und unter: Integrations Tools den Code erstellen (Auch Mischvarianten möglich bei gleicher Bannergrösse - z.B. Grafik und Textgrafik).
Momentan sind 468x60, 728x90 und 250x250 Grafik Banner und Text oder Text+Bild in Allen Formaten vorhanden.

Den erstellten Code Bitte unter Design/Widgets z.B. Im Seitenmenü platzieren oder sonstwo auf der Seite. Um die Werbung mittig zu zentrieren setzen Sie <center> vor und </center> nachdem Code ein.

Um die Werbeblocker der Seiten Betrachter zu kontrollieren schauen Sie unter: http://antiblock.org/

Auf die angegebene Paypal Email Adresse werden Verkäufe vergütet und zwar im 3 Phasen Modell:

-Anlauf mit Ratio 60% zu 40%
-danach 70% zu 30%
-Schliesslich 80% zu 20%

Immer zu Ihrem Günsten. Phasen werden im Mailing angekündigt.

Testen Sie es, es lohnt sich :o)


See also http://www.istok.de/

== Frequently Asked Questions ==

Warum gibt es nur 1 Kanal?

Ursprünglich gab es thematische Kanäle jedoch, nach Überlegung, ist es besser wenn mal 1 Werbung deplatziert gesehen wird als doch Vieles ungesehen bleiben würde. So z.B. kann sich auch eine Hausfrau für's Geldverdienen interessieren oder ein Webmaster sich um seine Gesundheit oder Liebes Probleme kümmern.

Ist die Werbung Conten Sensitiv?

Ja die Seiten werden kurz gescannt vor der Werbeinblendung, um nach Werbung, entsprechend vom Inhalt zu suchen.


See http://www.istok.de/
